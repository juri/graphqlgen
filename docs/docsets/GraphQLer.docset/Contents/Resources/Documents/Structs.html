  • Stringifier

    Stringifier is a protocol witness for stringifying GraphQL stuctures.

    There are implementations for the various structures as static properties in the appropriately typed extensions. The naming logic is that things called normal output their subject in more or less the only reasonable way, whereas things called compact produce a compact representation where a more expanded, indented, multi-line version would also be a reasonable choice.
